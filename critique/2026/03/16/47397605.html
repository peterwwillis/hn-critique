The article describes an analysis and simulation of the geography game Countryle, where the author built a solver to find the best strategy for guessing the target country using the game's feedback mechanisms. Main Points & Arguments

  • Countryle provides feedback on cardinal direction, continent, hemisphere, population, and average temperature after each guess.
  • The author built a modular solver using entropy to score guesses and filter possible countries based on feedback.

  • Libya was identified as the best starting country when combining all feedback channels equally, but other countries can outperform it depending on the metric used.
  • Simulations show the solver reaches the target in an average of 2.85 guesses with all feedback enabled.
  • The author notes limitations and suggests future improvements such as optimized weighting of feedback channels and better handling of population and temperature buckets.
