Unsloth Studio (unsloth.ai)

179 points by brainless 11 hours ago  | 39 comments critique  | original on HN

Summary

Unsloth Studio is an open-source, no-code web UI designed for training, running, and exporting various AI models locally on multiple operating systems. It supports a wide range of model types including text, vision, TTS audio, and embeddings, and offers features like dataset auto-creation, multi-GPU training, and model comparison. The platform is currently in beta and emphasizes privacy and offline use.

Main Points & Arguments

  • Unsloth Studio enables running GGUF and safetensor models locally on Mac, Windows, and Linux.
  • It supports training over 500 models with claimed 2x speed and 70% less VRAM usage without accuracy loss.
  • The platform allows no-code training by uploading various document types to auto-create datasets.
  • Supports multi-GPU training on NVIDIA GPUs and plans to support Apple MLX, AMD, and Intel soon.
  • Features include model export to formats compatible with llama.cpp and others, real-time training observability, and a model arena for side-by-side model comparison.
  • Unsloth Studio operates fully offline with token-based authentication to ensure data privacy.
  • The beta version is actively being improved, with acknowledgments to NVIDIA, Hugging Face, and other open-source contributors.
