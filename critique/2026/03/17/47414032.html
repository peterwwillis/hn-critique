Unsloth Studio (unsloth.ai)

348 points by brainless 24 hours ago  | 65 comments critique  | original on HN

Summary

Unsloth Studio is an open-source, no-code web UI designed to train, run, and export various AI models locally on multiple platforms. It supports running GGUF and safetensor models, no-code training from common document formats, multi-GPU training, and exporting models for use with popular inference engines. The platform emphasizes privacy by operating fully offline and offers features like data recipes, observability, and model comparison.

Main Points & Arguments

  • Unsloth Studio supports running and training over 500 models including text, vision, TTS audio, and embeddings on Windows, MacOS, Linux, and WSL.
  • The platform enables no-code training by auto-creating datasets from PDFs, CSVs, JSON, DOCX, and TXT files without needing pre-existing datasets.
  • It offers multi-GPU support, with ongoing improvements for Apple MLX, AMD, and Intel hardware compatibility.
  • Models can be exported in formats like GGUF and safetensors for use with llama.cpp, vLLM, Ollama, and LM Studio.
  • Unsloth Studio operates fully offline with token-based authentication to ensure user data privacy and security.
  • The beta version is actively being improved, with collaboration from NVIDIA and Hugging Face, and includes features like self-healing tool calling, auto inference tuning, and a model arena for side-by-side comparison.
