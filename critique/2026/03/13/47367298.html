Summary

Pyodide is a Python distribution that runs in the browser and Node.js using WebAssembly. It allows running Python code and packages, including those with C, C++, and Rust extensions, directly in web environments with a robust JavaScript-Python interface.

  • Pyodide is a port of CPython to WebAssembly/Emscripten.
  • It supports installing and running Python packages in the browser via micropip, including many scientific and general-purpose packages.
  • Pyodide provides a JavaScript-Python foreign function interface with support for error handling and async features.
  • Python code running in the browser via Pyodide has full access to Web APIs.
