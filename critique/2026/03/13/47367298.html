Summary

Pyodide is a Python distribution that runs in web browsers and Node.js environments using WebAssembly. It ports CPython to WebAssembly, enabling the use of Python packages, including those with C, C++, and Rust extensions, directly in the browser with a robust JavaScript-Python interface.

Main Points & Arguments

  • Pyodide is a port of CPython to WebAssembly/Emscripten, allowing Python to run in browsers and Node.js.
  • It supports installing pure Python packages from PyPi via micropip and includes many popular packages with native extensions like NumPy, pandas, SciPy, and Matplotlib.
  • Pyodide provides a comprehensive JavaScript-Python foreign function interface, supporting features like error handling and async/await.
  • Python code running in the browser via Pyodide has full access to Web APIs.
  • The project originated in 2018 at Mozilla as part of the Iodide project and is now a community-driven open-source project.
  • Pyodide is licensed under the Mozilla Public License 2.0 and has a large contributor base and active sponsorship.
