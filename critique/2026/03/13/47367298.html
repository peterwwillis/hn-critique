Summary

Pyodide is a Python distribution that runs in web browsers and Node.js environments using WebAssembly. It allows users to run Python code and install Python packages, including those with C, C++, and Rust extensions, directly in the browser with a JavaScript-Python interoperability layer.

Main Points & Arguments

  • Pyodide ports CPython to WebAssembly using Emscripten.
  • It supports installing pure Python packages via micropip and many packages with native extensions.
  • Includes scientific Python packages like NumPy, pandas, SciPy, Matplotlib, and scikit-learn.
  • Provides a robust JavaScript-Python foreign function interface with support for error handling and async/await.
  • Python code running in the browser has full access to Web APIs.
  • Pyodide was created in 2018 by Michael Droettboom at Mozilla as part of the Iodide project.
  • The project is open source, community-driven, and licensed under the Mozilla Public License 2.0.
