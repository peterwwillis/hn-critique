Summary

Regenerator 2000 is an interactive disassembler for the 6502 CPU, primarily targeting Commodore 8-bit computers. It features a text user interface (TUI) with modern conveniences such as cross-references, undo/redo, keyboard navigation, and integration with the VICE debugger.

Main Points & Arguments

  • Supports full 6502 instruction set including undocumented opcodes.
  • Provides side-by-side views of disassembly with hex dumps, sprites, bitmaps, charsets, and blocks.
  • Includes a debugger panel with live connection to VICE for breakpoints and stepping.
  • Supports multiple Commodore platforms like C64, C128, and Plus/4.
  • Can import various file formats including .prg, .crt, .d64, .d71, .d81, .t64, .vsf, .bin, .raw, and project files.
  • Exports assembly source compatible with assemblers such as 64tass, ACME, Kick Assembler, and ca65.
  • Offers project management features like saving/loading projects and auto-analysis of code/data regions.
  • Customizable display and navigation features including jump to address, cross-reference inspection, and history navigation.
  • Includes a Model Context Protocol (MCP) server for programmatic access.
  • Available for installation via Cargo and supports recommended modern terminals for best experience.
