Summary

The article introduces Attention Residuals (AttnRes), a novel modification to standard residual connections in Transformer models that uses learned, input-dependent attention over previous layer outputs to improve performance and training dynamics. It presents a variant called Block AttnRes to reduce memory usage and shows empirical results indicating performance gains on various benchmarks and better gradient distribution during training.

Main Points & Arguments

  • Standard residual connections in Transformers accumulate all layer outputs with fixed weights, which can dilute contributions and cause unbounded growth in hidden-state magnitudes.
  • AttnRes replaces fixed accumulation with softmax attention over preceding layer outputs, allowing selective, content-aware aggregation of earlier representations.
  • Block AttnRes partitions layers into blocks to reduce memory complexity while maintaining most benefits of full AttnRes.
  • Empirical results show AttnRes improves performance across multiple benchmarks including MMLU, GPQA-Diamond, BBH, TriviaQA, and code generation tasks.
  • AttnRes mitigates PreNorm dilution by keeping output magnitudes bounded and distributing gradient norms more uniformly across layers.
