Main Points & Arguments

  • The residual stream in transformers acts like a shared memory where different components read and write information in subspaces.
  • Attention mechanisms determine which token positions to read from, functioning as a soft address to select rows in the residual stream.

  • Induction heads enable transformers to predict repeated token patterns by composing information across layers using token:subspace addressing.
