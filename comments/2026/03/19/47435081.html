The file format (RAF, DNG) often is an acronym, but “raw” simply references the fact that file contains raw pixel data; it is not an acronym and therefore does not need all caps.

The mistake of “shouting” raw is forgivable because it is often perpetuated in the wild even by serious companies (Apple comes to mind[0]), but Adobe does use the correct spelling[1].

[0] It is fine when used as part of idiomatic spelling of their product (ProResRAW), but IIRC their developer docs do shout it when it simply references raw image data, which is a little ridiculous.

[1] https://helpx.adobe.com/camera-raw/digital-negative.html
